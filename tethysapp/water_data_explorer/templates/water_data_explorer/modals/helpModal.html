    The Hydroserver groups shows the different catalogs or groups that contains hydroservers, It allows the user to control the display of the Hydrogroup in the map with the chekbox. If unchecked, the Hydrogroup will not display in the map. Similarly, you can control the visualization of each hydroserver in the HydroGroup with the checkbox in the left of each hydroserver title. You can also press the zoom in Icon to zoom in to each one of the hydroserver.In addition, you can add or delete the hydroservers of any Hydrogroup with the menu right next to the name of the Hydrogroup. Note(The ability to add or deleted Hydroservers is only an ability for admin users, if you are not an admin user, you will not be able to see the button to delete or add hydroservers).

    Graphs and Hydroserver Information

    The graphs portion shows the Site name and description initially, then it shows the different variables in the selected site. The values of each variable area also shown. For instance, if the variable does not have any value it will show a 0 .The variables for the site are displayed in a table and also in a dropdown that promps the user to choose a variable to graph a time serie or whisker and box plot. In addition, there is two date pickers, so the user can select the beginning and ending date of the time series or whisker and box plot for such variable. The user is able to do this after pushing the update button. There is also graphs to see the distribution of the different variables: pie and bar graphs.
